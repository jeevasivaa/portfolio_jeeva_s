A modern, responsive personal portfolio website to present my profile, education, internship experience, projects, skills, certifications, achievements, and contact information.

## Live Overview
This portfolio includes:
- Clean and responsive layout
- Project cards with animated View modal
- Project descriptions with GitHub links
- Social and contact section
- Map-clickable location and direct email action
- Smooth scrolling and section highlight behavior

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Key Features
- Fully responsive design for desktop and mobile
- Interactive project modal with title, details, and GitHub button
- Smooth animations and hover interactions
- Contact panel with LinkedIn, GitHub, WhatsApp, and Instagram links
- Downloadable resume option
- Clean single-page portfolio structure
